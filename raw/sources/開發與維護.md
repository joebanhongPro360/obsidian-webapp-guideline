- component
    - 分爲
        - ui layer component
            - web-app/modules/components
                - menu (大區塊多頁複用的 UI 元件)
        - base component
            - web-common 內 storybook 做的元件
                - ButtonV2.jsx
    - 先從 web-common 內看有沒有共用，有的話先拿出來用
        - 有用到專案：
            - web-app
            - web-front
        - web-common 人看的話，先讀 storybook
        - web-common
            - 會先建立 js + css
            - 再由 ai 產 storybook
    - 如果 web-common 沒有的話，可以從 web-app/modules/components 查找有沒有 ui layer component
    - 如果不需要通用可以寫在當頁畫面直接使用
    - 可以與 ui 討論是否沿用舊的設計 web-common (storybook)，增加共用性
        - storybook 主要與 ui 溝通可以讀的文件
- css
    - styl (stylus) 專案起頭使用的。
    - 為了讓 web-app 與 web-common 內使用相同 css 套件，未來採用 scss
    - css 目前跟 component 放在一起直接使用。
        - Button.js
        - Button.module.scss
